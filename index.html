Le Langage C - Introduction

Le langage C est un langage de programmation de bas niveau très puissant et largement utilisé. Il est principalement utilisé pour développer des systèmes d'exploitation, des applications embarquées, des jeux vidéo, et d'autres logiciels nécessitant une gestion de bas niveau des ressources.

1. Types de données en C

Le langage C propose plusieurs types de données, dont :

  • int pour les entiers (ex : 1, 42, -7)
  • float pour les nombres à virgule flottante (ex : 3.14, 9.99)
  • char pour les caractères (ex : 'a', 'b', '1')
  • double pour les nombres à virgule flottante de double précision
        int a = 5;
        float b = 3.14;
        char c = 'A';

2. Déclaration et utilisation des fonctions

Les fonctions en C sont utilisées pour organiser le code en blocs réutilisables. Voici un exemple simple de fonction :

        void afficherMessage() {
            printf("Bonjour, bienvenue en C !");
        }

        int additionner(int x, int y) {
            return x + y;
        }

3. Structures de contrôle

Les structures de contrôle permettent de prendre des décisions ou de répéter des actions. Les principales structures sont :

  • if : exécute une action si une condition est vraie
  • for : répète une action un nombre défini de fois
  • while : répète une action tant qu'une condition est vraie
        if (a > b) {
            printf("a est plus grand que b");
        }

        for (int i = 0; i < 5; i++) {
            printf("%d", i);
        }

4. Tableaux

Un tableau est une collection de variables du même type, stockées dans des cases contiguës. Exemple :

        int tableau[5] = {1, 2, 3, 4, 5};

5. Pointeurs

Les pointeurs sont des variables qui stockent l'adresse mémoire d'une autre variable. Exemple :

        int x = 10;
        int *p = &x;  // p est un pointeur vers x
        printf("%d", *p);  // affiche la valeur de x via le pointeur p

6. Structures

Les structures permettent de regrouper différentes données sous une seule unité. Exemple :

        struct Personne {
            char nom[50];
            int age;
        };

        struct Personne p1 = {"Alice", 25};

7. Les commentaires

Les commentaires sont utilisés pour expliquer le code. Il existe deux types de commentaires :

  • // : commentaire sur une ligne
  • /* */ : commentaire sur plusieurs lignes
        // Ceci est un commentaire sur une ligne
        /* Ceci est un
           commentaire sur plusieurs lignes */

8. Compilateur C et Exécution

Pour exécuter un programme en C, vous devez d'abord le compiler avec un compilateur, tel que GCC. Une fois compilé, vous pouvez exécuter le programme via la ligne de commande :

        gcc programme.c -o programme
        ./programme
